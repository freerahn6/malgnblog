# 블로그 집필 10원칙 (작가 에이전트 필수 준수)

> 모든 글은 이 원칙을 통과해야 발행 후보가 된다. `_brief.md`·`_facts.md`·`_taxonomy.md`·`04_홍보_톤앤매너.md`와 함께 읽는다. 충돌 시 사실은 `_facts.md`(SSOT)가 최우선.

## 10원칙
1. **그래픽으로 스캔가능성 + SEO** — 글마다 시각요소를 넣어 심심하지 않게 + 검색 이점. 3종만 사용:
   - ⓐ **HTML `<table>`** (데이터·비교는 이미지 금지, `<caption>` + `<div style="overflow-x:auto">` 래퍼)
   - ⓑ **인라인 SVG 다이어그램/차트** (프로세스 흐름·단계·비교 막대 등). `<svg role="img">`에 `<title>`·`<desc>` 필수(접근성+색인). 색은 1~2색 절제, 텍스트 라벨 포함.
   - ⓒ **이미지 슬롯** `<figure><img src="./images/…webp" alt="서술형" width height><figcaption>…</figcaption></figure>` — 관리자 화면은 `C:\job\data\매뉴얼\…` 실제 스크린샷을 후속 자산패스에서 끼움.
     > ⚠️ **placeholder로 두면 안 된다.** 빌드는 본문 이미지를 `_deploy/figmap.json`에 해당 slug 항목이 있을 때만 data URI로 치환한다(`build.py`). 항목이 없으면 `/assets/…` 경로가 그대로 나가 **깨진 이미지로 발행된다.** 실제 이미지가 준비되지 않았다면 `<img>`를 아예 넣지 말고 표·SVG로 채울 것. (2026-07-23 실제 발생 → 발행 직전 제거)
   - 최소치: **글당 표 1개 이상 + SVG 다이어그램 1개 이상.** 이미지 슬롯은 **실제 이미지가 준비된 경우에만** 넣는다.
2. **리라이팅(중복 금지)** — 네이버 기존 글은 소재만 참고, **각도·구조·예시·문장을 전부 새로**. 문장 복붙 금지. 근거소스의 네이버 글을 읽되, 같은 순서·같은 표현을 피한다.
3. **검색의도 우선** — 타깃 키워드의 검색의도(정보/비교/방법/구매)에 정확히 답. 제목과 **첫 문단에 키워드 + 핵심 정답 한 문장**(발췌 스니펫·AI 답변 노림). 정의형 주제는 "OO란 …이다" 문장 포함.
4. **운영자 렌즈 + 교육 게이트** — AI·트렌드 주제도 결론은 "교육 운영 현장에서 뭘 하나"로 착지. 교육/학습과 무관하면 폐기.
5. **사실 SSOT** — `_facts.md` 검증 사실만. 금지: 교육효율 200%↑/관리시간 50%↓(마케팅 카피). 사용 가능 팩트: 누적 800개사 / 현재 약 375개 기관 / CSAP("국내 최초"·"상용 LMS 중 유일") / 동시접속 5,000명 / 데이터 4중 백업 / 16년. 규제·제도 수치는 단정 금지 → "HRD-Net·관할 고시/직능원 확인" 톤.
6. **E-E-A-T** — 추상론 대신 **매뉴얼 실제 기능명**·운영 디테일·출처. 규제/제도는 근거 링크 또는 "발행 시점 기준" 고지.
7. **스캔 구조** — H1은 front matter title로만(본문에 `#` 금지, `##`부터). 짧은 문단(3~4문장), 목록·요약박스·체크리스트 활용, 소제목에 키워드 변주.
8. **내부링크(클러스터)** — 같은/인접 클러스터 글로 **상대경로** 링크 2~3개(`/lms/…`, `/hrd/…`). slug는 `_taxonomy.md` 카테고리 규칙 준수. 실제 발행 slug만(미발행 글 링크는 계획된 slug로, 검수 시 정합).
9. **절제된 전환** — 정보·실무글: 글 끝 저압 CTA 1개(무료 상담/데모). 홍보글만 적극. 본문 중 광고 삽입 금지(신뢰장치는 04 규칙대로 글당 1~2회).
10. **일관 보이스·접근성·최신성** — "옆자리 15년차 교육운영 선배" 톤(형용사보다 기능·데이터·사례). 용어 첫 등장 영문 병기(LMS, LXP, SCORM, CSAP). 표 가로스크롤·이미지 width/height(CLS)·서술형 alt. 규제·수치엔 "발행 시점(YYYY-MM) 기준" 명시.
