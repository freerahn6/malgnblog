# 제품 지식 시트 — 위캔디오(WeCanDeo) · 영상 플랫폼 (Single Source of Truth)

> **용도**: 블로그팀(b1~b4)이 '자체 영상 플랫폼'·에듀테크·보안·진도연동 글을 쓸 때 **"위캔디오가 실제로 무엇인가"의 단일 출처**. 여기 없는 사양·수치는 본문에 쓰지 말 것. 작성 담당: 이채영(에듀테크·영상 플랫폼).
>
> **원본**: ① 온라인 지원센터 `https://support.wecandeo.com/` (2026-07 열람분) ② LMS 관리자 매뉴얼 내 위캔디오 로컬 문서 ③ 기존 글 `articles/why-own-video-platform.md`(로컬 매뉴얼 1차 증류분).

## 0. 레드라인 상속 (`_facts.md`에서) — 반드시 준수
- **미검증 수치 금지.** 이 시트에 근거로 적힌 수치만 사용하고, 그 외 성능·용량·언어 수 등은 창작·추정하지 말 것.
- **최상급·표방은 근거·스코프 병기.** "가장 쉬운 비디오 클라우드 플랫폼"은 위캔디오 **자사 슬로건**(support.wecandeo.com 표기)이므로 인용 시 '자사 표기' 전제로만. 검증 없는 "국내 1위/유일 영상 플랫폼" 표현 금지.
- **미동의 고객 실명·로고 금지.** 위캔디오 도입 고객 사례는 공개 동의 확인 전까지 업종 유형으로만.
- 아래 표기 규칙: `[온라인]` = support.wecandeo.com 문서에서 직접 확인 / `[로컬]` = LMS 관리자 매뉴얼(위캔디오 연동) 근거 / `[기존글]` = why-own-video-platform.md에서 이미 증류·사용 중 / `[확인필요]` = 원본 미열람 또는 값이 플랜·베타에 따라 변동될 수 있어 발행 전 재확인.

## 1. 제품 개요

- **위캔디오(WeCanDeo)** = (주)맑은소프트의 **자사 영상 클라우드 플랫폼**. 서버·CDN·인코딩 환경을 직접 구축하지 않고 영상 업로드·보안·스트리밍·통계를 **임대형(클라우드)으로 사용**하는 제품. [온라인][기존글]
- LMS 제품군 안에서의 위치: 맑은이러닝/범용 LMS의 **동영상플랫폼 구성요소**. `_facts.md` 제품군에 "위캔디오(WeCanDeo, 영상)"로 등재. LMS(학습관리)와 **진도율까지 연동**되는 형태로 함께 제공. [기존글]
- 자사 슬로건(자사 표기): "가장 쉬운 비디오 클라우드 플랫폼". [온라인]
- 제품 구성(패키지 관점): **VIDEO PACK**(VOD 업로드·인코딩·배포), **LIVE PACK**(라이브·웨비나), **PLAYER**(재생·연동), **VIDEOPACK AI**(자막·번역·요약), **Statistics**(통계), **OPEN API**. [온라인]
- 버전: 현재 **V4** 체계. AI 서비스는 "V4 비디오팩 AI"로 제공. [온라인]

## 2. 핵심 기능 사실 (기능 = 무엇 + 운영자에게 왜 + 인용 가능 문구)

### 2-1. 업로드 · 인코딩 [온라인][기존글]
- **무엇**: 대용량·멀티 파일 업로드, 인코딩 타입(화질/해상도 프로파일) 생성·설정, 구간 편집(트림), 인코딩 시 음성/오디오 볼륨 조절, 코덱 트랜스코딩(**AVC·HEVC**). [기존글=AVC·HEVC / 온라인=인코딩타입·볼륨조절·구간편집]
- **왜 중요**: 원본을 올리면 다양한 화질로 자동 변환돼, 수강생 회선·기기에 맞춰 끊김 없이 재생. 운영자가 서버·인코더를 직접 다루지 않아도 됨.
- **인용 가능 문구**: "위캔디오는 대용량·멀티 업로드와 AVC·HEVC 트랜스코딩을 지원해, 서버 없이도 원본 영상을 다화질로 배포할 수 있다."
- [확인필요]: 지원 입력 포맷 상세 목록, 최대 파일 용량 — 원본 수치 미확인.

### 2-2. 배포 패키지(Distribution) · 폴더 연동 [온라인]
- **무엇**: 업로드·인코딩한 영상을 **배포 패키지(배포 소스)** 단위로 묶어 서비스에 노출. 동영상 폴더 연동, 메타데이터(제목·설명 등) 수정, 플레이리스트 구성, 썸네일/미리보기 지원.
- **왜 중요**: 영상 하나가 '재생 가능한 서비스'가 되는 단위. 배포 패키지에서 보안(DRM)·플레이어·콜백을 함께 설정 → LMS 연동의 출발점.
- **인용 가능 문구**: "위캔디오는 영상을 배포 패키지 단위로 관리하며, 이 단계에서 보안·플레이어·시청기록 연동 설정이 함께 이뤄진다."

### 2-3. 플레이어(PLAYER) — Embed · iframe · 커스텀 [온라인][기존글]
- **무엇**: **HTML5 기반** 플레이어. **iframe/Embed 코드 자동 생성**으로 웹·앱에 삽입. 컨트롤 영역 노출 설정, 시작 시점(특정 타임스탬프) 지정, 플레이리스트, 자막·챕터·썸네일 미리보기. [온라인] / 임베드 코드 자동 생성 [기존글]
- **브라우저 자동재생 정책 대응**, **접근성 제공 정책**(플레이어 접근성) 명시. [온라인]
- **왜 중요**: 플레이어 색상·로고·컨트롤을 우리 브랜드로 구성 → '유튜브 화면'이 아닌 자사 브랜드 경험. 도메인·서비스에 그대로 임베드.
- **인용 가능 문구**: "위캔디오 플레이어는 HTML5 기반으로 iframe 임베드 코드를 자동 생성하며, 컨트롤·자막·챕터·접근성까지 설정할 수 있다."
- [확인필요]: 플레이어 SDK(Player SDK) 커스터마이징 상세 — OSS/SDK 샘플 존재만 확인, 세부 미열람.

### 2-4. 보안 — Multi-DRM · SecureDRM · 워터마크 · 동시시청 제한 [온라인][기존글]
- **Multi-DRM**: **Widevine**(Chrome)·**PlayReady**(Edge)·**FairPlay Streaming**(Safari) 3종을 지원해, 별도 플러그인 없이 일반 브라우저에서 DRM 콘텐츠 재생. 콘텐츠를 암호화하고 정당한 사용자에게만 복호화 키 제공. 보안 전문사 **잉카엔트웍스(Inka Networks)**와 연계. [온라인]
- **SecureDRM**(배포 패키지 부가 옵션): 적용 시 보안 플레이어 자동 사용, 출력 포맷 MP4 고정. 적용 시 일부 콘솔 UI(미리보기·플레이어설정·광고·자막)는 제외. **V4 + Enterprise 플랜 이상 + 별도 옵션 계약** 필요. [온라인]
- **사용자 워터마크**: 시청자 식별용 워터마크 설정 가능(재유출 추적). [온라인][기존글]
- **동시시청 제한(중복접속 차단)**: 플레이어 동시시청 제한 설정으로 계정 공유·동시 재생 차단. [온라인][기존글]
- **녹화방지**: 화면 녹화 방지. [기존글]
- **왜 중요**: 유료·대외비·법정의무 교육 영상의 링크 유출·재배포·계정 공유를 막는 핵심. 유튜브가 못 하는 '접근·콘텐츠 소유권'의 실체.
- **인용 가능 문구**: "위캔디오는 Widevine·PlayReady·FairPlay 멀티 DRM과 사용자 워터마크, 동시시청 제한을 제공해 유료·사내 교육 영상의 유출을 통제한다."
- [확인필요]: 우클릭/다운로드 차단 등 개별 방어 항목의 정확한 사양 — 문서에 명시 없음.

### 2-5. 자막(Caption) [온라인]
- **무엇**: 자막 삽입 및 변환(수동), WebVTT(VTT) 형식.
- **인용 가능 문구**: "위캔디오는 VTT 자막 삽입·변환을 지원한다." (AI 자동생성은 2-7 참조)

### 2-6. 통계(Statistics) [온라인][기존글]
- **무엇**: 재생 시간·재생 위치(구간) 데이터, 재생/사용량 통계. 회원·구간·디바이스·지역별 시청 분석 리포트. [기존글=구간·디바이스·지역 / 온라인=재생시간·위치·사용량]
- **왜 중요**: '누가 어디까지 봤나'가 곧 이수 근거·콘텐츠 개선 데이터. 자체 플랫폼만의 데이터 소유권.
- **인용 가능 문구**: "위캔디오는 재생 구간·디바이스·지역 단위 시청 통계를 제공한다."
- [확인필요]: Statistics API의 구체 지표·집계 단위 상세.

### 2-7. VIDEOPACK AI (자막생성 · 번역 · 요약) [온라인]
- **AI 자막생성**: VOD의 음성을 자동 인식(STT)해 WebVTT 자막 생성. **언어 자동 감지**(사전 선택 불필요). 지원 **최대 99개 언어**, 영상 **최대 180분**. [온라인]
- **AI 자막번역**: 기존 VTT 자막을 다른 언어로 번역(타임코드 유지). 지원 **75개 언어**. [온라인]
- **AI 요약**: VTT 자막을 분석해 핵심 요약 텍스트 생성. 지원 **29개 이상 언어**, 건당 과금. [온라인]
- **처리 방식**: 실시간 아님(비동기). 목표 24시간 이내 처리(SLA 보장 아님). 실패 작업은 과금 없음. [온라인]
- **성격**: 사람의 검수를 보조하는 기능 — 법률·의료 등 고위험 콘텐츠는 배포 전 검수 권고. [온라인]
- **왜 중요**: 강의 영상의 자막·다국어·요약을 사람 없이 1차 생성 → 접근성·글로벌 교육·복습자료 제작 시간을 크게 단축.
- **인용 가능 문구**: "위캔디오 AI는 음성을 인식해 자막을 자동 생성하고, 다국어 번역과 요약까지 제공한다(언어 자동 감지)."
- [확인필요]: **언어 수(99/75/29)·180분·무료 체험량(생성·번역 600분, 요약 20건)·플랜별 과금**은 플랜/베타 정책에 따라 변동. 발행 시 정확 수치는 지원센터 재확인 후 사용하고, 애매하면 "다국어 자막 자동생성·번역·요약을 지원한다" 수준의 정성 표현 권장. (베타 무료 공지와 유료 플랜 안내가 문서상 병존 → 요금/무료 여부 단정 금지.)

### 2-8. V4 AI (정식 출시) [온라인][확인필요]
- 지원센터 Announcements에 **V4 비디오팩 AI 정식 출시** 공지 존재(2-7의 3종 AI 서비스가 V4 체계로 제공). 정식 출시 **일자·플랜 구성(Free/구독 Starter·Standard·Pro/Enterprise)** 은 공지 원문 재확인 후 인용. (Announcements 상세 페이지는 이번 열람에서 직접 접근 불가 → [확인필요].)

### 2-9. LIVE PACK · 웨비나 [온라인][확인필요]

## 3. LMS 연동 흐름 (교육 운영의 핵심 경로)

업로드 → 인코딩 → **배포 패키지** → **플레이어 연동(임베드)** → **진도율 콜백** → LMS 진도율·이수 집계. [로컬 원문 직접 확인]

> ※ 아래 1~5는 로컬 위캔디오 매뉴얼 6개 페이지(동영상 업로드·진도율 콜백 설정·플레이어 추가 및 연동·플레이어 기본 설정·커스텀 설정·관리자 및 알림 설정) **원문을 직접 열람해 확정한 1차 근거**다.

1. **업로드·인코딩**: 위캔디오 비디오팩 로그인 → 원본 업로드 시 **인코딩이 함께 진행**. **기본 인코딩 타입 = 720p·540p·360p HLS**(타입 추가 가능). 업로드 시 **보관함 폴더 지정 + 배포 패키지 지정**이 필요하며, **배포 패키지를 지정하지 않으면 인코딩되지 않는다**(원본 그대로면 트래픽 과다). [로컬]
2. **배포 패키지 생성·설정**: 영상을 배포 소스로 묶고, 여기서 보안(DRM/워터마크)·플레이어·콜백을 함께 설정. [온라인][로컬]
3. **플레이어 추가·연동**: 미디어 보관함 → **플레이어**에서 LMS 연동 플레이어 선택·적용하거나 **[플레이어 만들기]로 커스텀 플레이어 생성**. 배포 패키지 동영상에 **[플레이어 설정 적용하기]**로 연결. [로컬]
   - **플레이어 기본 옵션**(교육용 핵심): 사이즈, 자동시작, **seek 금지(재생바 위치 변경 금지)**, 시작 시간, 썸네일 미리보기, 화면 클릭 재생/일시정지. → **seek 금지는 '건너뛰기 방지=성실 수강 증빙'의 실무 근거.** [로컬]
   - **커스텀 설정**: 썸네일·**워터마크**·**자막**·북마크·헤더·스타일·구글 애널리틱스 연동을 플레이어 단위로 설정. [로컬]
4. **진도율 콜백 설정**: 배포 → 배포패키지 → **배포 설정** 탭에서 **Playback Callback 주소** `https://{사이트도메인}/wecandeo/check_api.jsp` 입력 + **전송주기 10초**로 설정 → 학습자의 시청 진행이 LMS 진도율로 자동 반영. **콜백을 설정하지 않으면 LMS 진도율 체크가 되지 않음** → 교육용은 필수 설정. [로컬]
5. **관리자·알림 설정**: 위캔디오 콘솔에서 **관리자 계정 등록/삭제**, **사용량 이메일 알림** 설정(적용하기). **무제한 요금제는 스토리지 사용량 알림만 수신 권장.** [로컬]

> **CDN(스트리밍)**: AWS·LG U+·CDNetworks·NHN Cloud 등 CDN을 통한 배포. [기존글] — 정확한 현재 제휴 CDN 목록은 발행 전 재확인 권장 [확인필요].

### why-own-video-platform.md 보강 포인트
- 기존 글의 보안 표(DRM·녹화방지·워터마크)에 **Multi-DRM 3종 명칭(Widevine/PlayReady/FairPlay)** 과 **동시시청 제한**을 근거로 추가 가능.
- 기존 글 '지원 기능' 불릿에 **VIDEOPACK AI(자막생성·번역·요약)** 를 에듀테크 강점으로 추가 가능(단, 언어 수·요금은 정성 표현).
